## Install

> [!IMPORTANT]
> The Helm chart does not handle opening Gluetun firewall ports automatically. You are responsible for opening the ports you need.
> For details, see [Firewall Environment Variables](https://github.com/qdm12/gluetun-wiki/blob/main/setup/options/firewall.md#environment-variables).

```console
helm install qbittorrent oci://ghcr.io/sudo-kraken/helm-charts/qbittorrent
```

Verify the signature with [cosign](https://docs.sigstore.dev/cosign/system_config/installation/) :

```console
cosign verify ghcr.io/sudo-kraken/helm-charts/qbittorrent:5.1.3 --certificate-identity=https://github.com/sudo-kraken/helm-charts/.github/workflows/releases.yml@refs/heads/main --certificate-oidc-issuer=https://token.ac
tions.githubusercontent.com
```
